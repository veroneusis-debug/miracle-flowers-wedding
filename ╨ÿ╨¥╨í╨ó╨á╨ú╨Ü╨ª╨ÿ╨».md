# 🌸 Miracle Flowers — Инструкция по настройке админки

## Что внутри этой папки

```
miracle-flowers-cms/
├── index.html          ← Лендинг (главная страница)
├── netlify.toml        ← Настройки Netlify
├── logo.png
├── logo.jpg
├── photo.jpg
├── certificate.jpg
├── images/             ← Сюда загружаются новые фото через админку
├── content/            ← Контент лендинга в JSON-файлах
│   ├── hero.json           (главный экран)
│   ├── how_it_works.json   (как это работает)
│   ├── certificates.json   (карточки депозитов)
│   ├── callback.json       (блок "Мы на связи")
│   └── footer.json         (подвал)
└── admin/
    ├── index.html      ← Страница входа в админку
    └── config.yml      ← Конфигурация полей
```

---

## Шаг 1 — Создать аккаунт на GitHub (если нет)

1. Зайти на https://github.com
2. Нажать **Sign up** и зарегистрироваться
3. Создать новый репозиторий: нажать **+** → **New repository**
   - Название: `miracle-flowers-wedding`
   - Тип: **Public**
   - Нажать **Create repository**

---

## Шаг 2 — Загрузить файлы на GitHub

В своём репозитории нажать **uploading an existing file** и загрузить ВСЕ файлы и папки из этого архива.

Или использовать GitHub Desktop (приложение) — там всё визуально.

---

## Шаг 3 — Подключить к Netlify

1. Зайти на https://netlify.com → **Sign up** (через GitHub)
2. Нажать **Add new site** → **Import an existing project**
3. Выбрать **GitHub** → выбрать репозиторий `miracle-flowers-wedding`
4. Нажать **Deploy** — сайт развернётся автоматически

---

## Шаг 4 — Включить Git Gateway (для работы CMS)

1. В Netlify открыть свой сайт → вкладка **Site configuration**
2. Перейти в **Identity** → нажать **Enable Identity**
3. Там же найти **Git Gateway** → нажать **Enable Git Gateway**
4. Вернуться в **Identity** → **Invite users** → ввести свой email

На email придёт письмо — нажать **Accept the invite**. Это создаст ваш аккаунт для входа в админку.

---

## Шаг 5 — Войти в админку

Открыть: `https://ВАШ-САЙТ.netlify.app/admin`

Ввести email и пароль, который вы установили при принятии приглашения.

---

## Как редактировать контент

В админке слева вы увидите разделы:

- **🌸 Главный экран** — имена, текст, фото, кнопка
- **✦ Как это работает** — заголовки и описания трёх шагов
- **🎫 Сертификаты** — суммы и теги карточек депозитов
- **📞 Блок обратного звонка** — тексты и номер телефона
- **🌿 Футер** — адрес, ссылки, юридическая информация

### Как поменять фото:

1. Открыть **Главный экран**
2. Нажать на поле **Фото молодожёнов**
3. Нажать **Choose an image** → **Upload**
4. Выбрать новое фото с компьютера
5. Нажать **Save** — изменения появятся на сайте через ~30 секунд

---

## Частые вопросы

**Когда изменения появятся на сайте?**
Обычно через 20–60 секунд после нажатия Save.

**Можно ли работать с телефона?**
Да, админка адаптирована для мобильных.

**Что если что-то сломалось?**
В Netlify есть история деплоев — можно откатиться к прошлой версии.
